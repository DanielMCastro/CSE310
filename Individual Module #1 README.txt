# Overview

I made this game to learn more about python, and how to create a video game, and the process behind it.

This is a simple game where you control a charecter that jumps over monsters that come towards you. If you fail to jump over, the game ends.

I wrote this game to learn more about game development

Game demo video: https://youtu.be/TxXq8r6CaMA
